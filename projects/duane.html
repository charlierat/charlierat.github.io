Project Overview

Developed a real-time computer vision application using Python, OpenCV, NumPy, and Tkinter to simulate the visual effects of Duane syndrome, a rare eye movement disorder that causes double vision and peripheral vision loss.

The application processes live camera feeds by applying advanced image processing techniques including Gaussian filtering, pixel shifting, and mask-based blending to accurately recreate the asymmetric visual distortions experienced by patients with this condition.

This educational tool demonstrates proficiency in medical simulation software development, real-time image processing algorithms, and GUI design while serving as an accessibility awareness resource for healthcare professionals and educators.

Medical Context

About Duane Syndrome

Duane syndrome is a rare eye movement disorder that affects the lateral rectus muscle, causing difficulty moving the eye outward and often resulting in double vision (diplopia), reduced peripheral vision on the affected side, and compensatory head turning to maintain clear vision.

# Create exponential falloff mask for effect intensity def create_focus_mask(self, frame): y, x = np.ogrid[:height, :width] center_x, center_y = self.focus_point # Calculate distance from focus point distance = np.sqrt((x - center_x)**2 + (y - center_y)**2) # Create exponential falloff mask = np.exp(-distance / self.focus_area_size) return np.clip(mask, 0, 1)
