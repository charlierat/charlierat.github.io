Expense Tracker

Full-stack mobile application for personal finance management featuring real-time synchronization, secure authentication, and comprehensive financial tracking capabilities across iOS and Android platforms.

Technology Stack

React Native Expo JavaScript Node.js Express.js PostgreSQL Redis Clerk Auth

Project Overview

Developed a comprehensive full-stack mobile expense tracking application using React Native, Expo, and JavaScript with custom CSS styling, featuring secure Clerk authentication with email verification and a comprehensive 5-screen interface for financial management.

Built a robust backend using Node.js, Express.js, and PostgreSQL hosted on Neon cloud platform, implementing RESTful APIs with Redis-based rate limiting for secure transaction management and real-time balance calculations.

The application includes advanced features such as pull-to-refresh functionality built from scratch, one-tap transaction deletion, and CRUD operations for income/expense entries with automatic balance updates.

Key Features

  • Secure user authentication with email verification using Clerk
  • Real-time financial dashboard with balance calculations and summaries
  • Complete transaction management with categorization and validation
  • Cross-platform compatibility for both iOS and Android devices
  • Pull-to-refresh functionality and automatic data synchronization
  • Rate limiting and security measures with Redis implementation
  • Cloud deployment architecture enabling real device testing
  • Responsive design with smooth animations and loading states

Technical Implementation

Database Schema

CREATE TABLE transactions( id SERIAL PRIMARY KEY, user_id VARCHAR(255) NOT NULL, title VARCHAR(255) NOT NULL, amount DECIMAL(10,2) NOT NULL, category VARCHAR(255) NOT NULL, created_at DATE NOT NULL DEFAULT CURRENT_DATE )

Rate Limiting

const ratelimit = new Ratelimit({ redis: Redis.fromEnv(), limiter: Ratelimit.slidingWindow(100, "60 s"), })

Implemented comprehensive security measures including rate limiting (100 requests/minute), input validation, error handling, and automated health checks with cron jobs to prevent server sleep on free hosting tiers.
